# base-parent
基础的springboot项目封装，base springboot project

### 启动参数

```text
-Ddomain=demo
-Dspring.profiles.active=dev
-Dspring.application.name=demo
-Dlog.file=D:\log\springboot\xx
-Dserver.port=8082
-XX:+HeapDumpOnOutOfMemoryError
-XX:HeapDumpPath=D:/java/logs/admin/oom

```
